# moniterinstance

## install
```
$ git clone https://github.com/ritarock/moniterinstance.git
$ cd moniterinstance/
$ go install
```

## Usage
```
NAME:
   moniterinstance - moniter ec2 instance

USAGE:
   moniterinstance [global options] command [command options] [arguments...]

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --profile value  set aws profile (default: "default")
   --term value     set StartTime (default: 60)
   --help, -h       show help (default: false)
```
